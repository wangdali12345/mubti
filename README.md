### 核心算法
```javascript
// MBTI类型计算
function calculateMBTI() {
    const dimensions = {
        EI: scores.E >= scores.I ? 'E' : 'I',
        SN: scores.S >= scores.N ? 'S' : 'N',
        TF: scores.T >= scores.F ? 'T' : 'F',
        JP: scores.J >= scores.P ? 'J' : 'P'
    };
    return dimensions.EI + dimensions.SN + dimensions.TF + dimensions.JP;
}
```

## MBTI维度说明

### **E - 外向 (Extraversion) vs I - 内向 (Introversion)**
- **E**：从外部世界获得能量，喜欢社交和互动
- **I**：从内心世界获得能量，享受独处和思考

### **S - 实感 (Sensing) vs N - 直觉 (Intuition)**
- **S**：关注具体事实和实际经验，注重现实
- **N**：关注可能性和潜在含义，注重未来

### **T - 思考 (Thinking) vs F - 情感 (Feeling)**
- **T**：基于逻辑和客观分析做决定
- **F**：基于价值观和人际感受做决定

### **J - 判断 (Judging) vs P - 感知 (Perceiving)**
- **J**：喜欢有计划、有组织的生活方式
- **P**：喜欢灵活、开放的生活方式
