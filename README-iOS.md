# 📱 苹果手机无法访问解决方案

## 问题原因

苹果iOS系统对本地 `file://` 协议有严格的安全限制：
- 禁止加载外部资源（如Google字体）
- 限制JavaScript执行
- 不支持某些CSS特性
- 安全策略更加严格

## 🚀 快速解决方案

### 方法一：使用Python服务器（推荐）

1. **运行Python服务器**
   ```bash
   # 在项目目录中运行
   python ios-server.py
   ```

2. **手机访问**
   - 确保iPhone和电脑在同一WiFi
   - 电脑运行服务器后显示IP地址
   - 在iPhone浏览器中访问: `http://[电脑IP]:8080`

### 方法二：使用Node.js服务器

1. **运行Node.js服务器**
   ```bash
   node server.js
   ```

2. **访问地址同上**

### 方法三：一键启动（Windows）

双击运行 `ios-fix.bat` 文件，会自动检测并启动合适的服务器

## 🔧 手动设置方法

### 1. 查找电脑IP地址
- **Windows**: 打开命令行，输入 `ipconfig`
- **Mac**: 打开终端，输入 `ifconfig`
- 找到类似 `192.168.1.x` 的地址

### 2. 手机访问
- 打开iPhone的Safari浏览器
- 输入: `http://[电脑IP]:8080`
- 例如: `http://192.168.1.100:8080`

## 🛠️ 故障排除

### 仍然无法访问？
1. **检查防火墙设置**
   - Windows防火墙可能阻止端口8080
   - 临时关闭或添加例外

2. **检查网络连接**
   - 确保设备在同一WiFi
   - 尝试关闭移动数据

3. **检查端口占用**
   ```bash
   # 检查端口是否被占用
   netstat -ano | findstr :8080
   ```

### 字体显示问题
- 可能是网络限制导致Google字体无法加载
- 服务器已添加CORS支持

### 功能异常
- 检查浏览器控制台错误信息
- 尝试清除浏览器缓存
- 确保JavaScript已启用
