# MBTI 16型人格测试 - 苹果风格版本

一个完全按照苹果官网设计语言重构的MBTI人格测试网页应用。

## 🍎 苹果风格设计特色

### 视觉设计
- **极简主义**：简洁的白色背景，大量留白空间
- **Inter字体**：采用苹果系统同等级的现代无衬线字体
- **蓝色主题**：使用苹果标志性的 #0071e3 蓝色作为主色调
- **卡片式布局**：圆角边框，微妙阴影，层次分明

### 交互体验
- **流畅动画**：使用苹果推荐的 cubic-bezier 缓动函数
- **微交互**：悬停效果、点击反馈、选中状态
- **页面切换**：淡入淡出和平滑位移过渡效果
- **进度指示**：简洁的进度条，带有光影动画效果

### 响应式设计
- **精准适配**：针对 iPhone、iPad、Mac 不同设备优化
- **触摸优化**：移动端友好的触摸区域和手势支持
- **系统适配**：支持深色模式和用户偏好设置
